# ServeIQ v0.3

نسخة محسنة من نظام حمسة مجالس.

## التحسينات
- إصلاح إرسال الطلب على Safari وiPhone.
- استخدام نموذج form لإرسال الطلب بشكل موثوق.
- إنشاء رقم تعريفي احتياطي إذا لم يدعم المتصفح randomUUID.
- رسائل خطأ واضحة عند تعذر الحفظ.
- حفظ الطلبات محليًا تحت مفتاح v0.3.

## التشغيل
```bash
npm install
npm run dev
```
